# 某某计划管理平台 

  极简的Element UI 学习和熟练vue 框架 \
  自己写一套看板展示组件,支持动态拖拽以及添加删除,还是有点技术难度的\  
下载后\
       npm install    \
        npm run   \
   1)先点一下用户权限和流程管理,首页和计划进度才有数据,首页和计划计划进度都是动态数据来源都是用户权限和流程管理  \
   2）当然在用户权限和流程管理也可以新增数据和删除数据,这里数据缓存采用的localstore  \
   3）不多说看代码你就知道  \
